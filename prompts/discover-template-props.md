Read first:
  guardrails/guardrails.md

Then do the following — READ ONLY, no edits to any template file:

TASK:
For each of these templates:
  src/templates/template_classic/
  src/templates/template_modern/
  src/templates/template_premium/

1. Find the entry point (index.tsx or HomePage.tsx or similar root file)
2. Read what top-level props the component accepts
3. For each section, document:
   - The prop name passed to that section
   - The type/shape expected
   - Whether it is optional or required

OUTPUT:
Create one file per template at:
  guardrails/template-props/template_classic.md
  guardrails/template-props/template_modern.md
  guardrails/template-props/template_premium.md
