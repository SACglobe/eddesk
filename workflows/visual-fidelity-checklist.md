# Visual Fidelity Checklist (Next.js Templates)

This checklist must pass 100% before a template is accepted.

## A. Layout Structure
- [ ] Overall page height matches source
- [ ] Section order is identical
- [ ] Section spacing (margin / padding) matches
- [ ] Containers align correctly at all breakpoints

## B. Typography
- [ ] Font family matches source
- [ ] Font size matches source
- [ ] Font weight matches source
- [ ] Line height matches source
- [ ] Letter spacing matches source

## C. Colors
- [ ] Background colors match
- [ ] Text colors match
- [ ] Border colors match
- [ ] Hover / active states match

## D. Images & Media
- [ ] Image dimensions match
- [ ] Aspect ratios preserved
- [ ] Image alignment matches
- [ ] Icons match size and position

## E. Components
- [ ] Headers match exactly
- [ ] Buttons match size, shape, and spacing
- [ ] Cards / sections match alignment
- [ ] Navigation behaves identically

## F. Responsiveness
- [ ] Desktop (≥1280px) matches
- [ ] Tablet (768–1024px) matches
- [ ] Mobile (≤480px) matches
- [ ] No layout shifts between breakpoints

## G. Behavior
- [ ] Hover states match
- [ ] Animations (if any) match
- [ ] Scroll behavior matches

## Verdict
- [ ] 100% visually identical
- [ ] NO “close enough”
